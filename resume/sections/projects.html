專案

VenvManager

Python + Tkinter 的 Windows 虛擬環境管理工具：建立/刪除 venv、安裝套件（單個或由檔案）、列出套件與 Python 版本，並能執行 activate.bat。

Tech
Python 3、Tkinter
重點
GUI 操作、requirements 支援、封裝嘗試

DiscordStreamerAlert

Discord 機器人整合 Twitch API：監控串流主開播即通知頻道；支援自訂指令（SQLite 儲存）。

Tech
Python 3.11+、Discord/Twitch API、SQLite
功能
!addstreamer、!removestreamer、!liststreamers、!addcommand、!removecommand
配置
config.yml（含 client_id/secret、token_url 等）

Google Route Planner on Flask

以 Flask 包裝 Google Maps 路線規劃：輸入多筆起訖點，回傳距離/時間與地圖結果。

Tech
Flask、Google Maps API、PyYAML
部署
`pip install -r requirements.txt`，`python app.py` → http://localhost:5000/
設定
Config.yml 需填入 Google Maps API 金鑰

Simple Chat Application

Bootstrap 5 聊天 UI 範例：快速帶入專案作為訊息介面起手式，附截圖與元件來源連結。

Tech
Bootstrap 5、純前端
亮點
輕量、易嵌入、即取即用的聊天版型
